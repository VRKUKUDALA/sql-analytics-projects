#  SQL Analytics Projects

## Overview
This repository contains hands-on SQL analytics projects designed to solve
real-world business and analytical problems using relational databases.

The projects demonstrate proficiency in writing efficient, readable, and
scalable SQL queries using joins, subqueries, Common Table Expressions (CTEs),
and window functions.

---

## Datasets Used

### MavenMovies Database
Used for business-focused analytics including:
- Rental trends and seasonality
- Film popularity and genre demand
- Store and staff performance
- Customer behavior and revenue analysis

### ClassicModels Database
Used to demonstrate advanced SQL concepts such as:
- Subqueries and correlated queries
- Common Table Expressions (CTEs)
- Window functions and moving averages
- Customer-level spend analysis

---

## Objectives
- Translate business questions into SQL-based analytical solutions
- Identify trends, patterns, and performance drivers
- Apply advanced SQL techniques to real datasets
- Demonstrate job-ready SQL query design

---

## Tools Used
- SQL (MySQL)
- Advanced SQL features:
  - JOINs and Subqueries
  - CTEs
  - Window Functions 
  - Aggregations and analytical calculations

---

## Repository Structure
- `mavenmovies/` → Business analytics using the MavenMovies database
- `classicmodels/` → Advanced SQL analytics using the ClassicModels database
